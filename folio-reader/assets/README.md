# Assets

Place the following image files in this directory:

- `icon.png` — 1024×1024 app icon
- `splash.png` — 1284×2778 splash screen image
- `adaptive-icon.png` — 1024×1024 Android adaptive icon foreground

All images should have a dark background (#0f0f14) to match the app theme.

You can generate simple placeholder icons using any image editor or online tool.
Expo will handle resizing for all device sizes automatically.

## Quick placeholder

If you just want to run the app immediately, you can generate placeholders with:

```bash
# Install Expo CLI if not already installed
npm install -g expo-cli

# Use expo-doctor to check your setup
npx expo-doctor
```

Or simply copy any 1024×1024 PNG and name it `icon.png`, `splash.png`, and `adaptive-icon.png`.
